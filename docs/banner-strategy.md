# Banner Strategy

## Purpose

The banner is no longer a branding image.

It becomes a conversion module.

## Banner Types

### Type 1

Product Category Banner

Purpose:

- Explain research coverage
- Drive report exploration

CTA:

- Explore Reports
- Subscribe

---

### Type 2

Featured Report Banner

Purpose:

- Promote premium reports
- Increase report sales

CTA:

- View Report
- Request Sample

---

### Type 3

Campaign Banner

Purpose:

- Webinar
- Whitepaper
- Sponsored Report

CTA:

- Register Now
- Learn More
